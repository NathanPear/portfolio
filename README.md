# portfolio
Static version of Nathan Peredery's technical portfolio, rebuilt from the Google Sites content for GitHub Pages hosting.

Site tree:
- `index.html`
- `software.html`
- `mechanical.html`
- `electronic.html`
- `integrated.html`
- `projects/`

Content source:
- `data/projects.json`

Build:
- `npm run build`

The build script reads the JSON registry and injects project cards into the tagged sections of the static HTML pages.

Open `index.html` directly or publish the repo with GitHub Pages.
