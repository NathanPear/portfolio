PROJECT 3 / ELECTRONIC

Student Number FSM

A logic design project that uses JK flip-flops and custom encoding to drive a 7-segment display through a repeated-digit sequence.

  • Role: digital logic design and debugging
  • Focus: state encoding and stable transitions
  • Constraint: repeated digits without incorrect looping

Project Details

Problem and Context

The goal was to display a nine-digit student number on a 7-segment display, including repeated digits that could not be treated as a simple linear state chain.

That made the project a good test of sequential logic, encoding discipline, and careful transition planning.

Scope and Constraints
  • Use JK flip-flops for the state machine
  • Keep the architecture NAND-only where possible
  • Support manual reset
  • Drive a readable 7-segment output

Approach

What I Built
  • A 4-bit state controller for the display sequence
  • A custom encoding scheme to distinguish repeated digits
  • MSOP expressions converted into NAND-only logic
  • Synchronous transitions with a manual reset path
Cross-Disciplinary Read

This project is easy to understand from both a hardware and a systems perspective. The circuit has to be small enough to be elegant, but also predictable enough to behave correctly every time it resets and advances.

That makes it useful evidence for any reviewer who cares about logic design, state management, or debugging under constraints.

Outcome

Result

The final FSM handled the repeated-digit sequence correctly and produced a stable display path for the student number.

It also demonstrated how a compact state encoding can keep a logic design manageable without sacrificing correctness.

Takeaways
  • State encoding is part of the design, not just a bookkeeping step
  • Reset behavior matters as much as the nominal sequence
  • Readable case studies should explain the logic in plain language
